All of the assignments are done in a group of 3. It is related to a game design project.

Assignment 1 requires us to draw a UML diagram representing all the interacting elements of our own design with the provided game engine, all while fulfilling the requirements stated in the assignment specifications.

Assignment 2 requires us to implement the game logic following the UML diagram in Assignment 1.

Assignment 3 introduces new elements to the game, and we have to implement them into the base game from Assignment 2 without breaking the original logic flow.
